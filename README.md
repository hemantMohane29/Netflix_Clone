# Netflix Clone

A responsive, pixel-perfect frontend clone of Netflix built with **HTML5** and **CSS3**. This project replicates the sleek design and interactive elements of the popular streaming platform's homepage.

![Netflix Clone Banner](https://via.placeholder.com/1200x400/141414/FFFFFF?text=NETFLIX+CLONE)  
*(Replace with actual screenshot once added to assets)*

## ✨ Features

- **Fully Responsive Design** — Optimized for desktop, tablet, and mobile devices
- **Modern Dark UI** — Authentic Netflix-style black/red theme
- **Interactive Navigation** — Hover effects, smooth transitions, and dropdown menus
- **Hero Banner** — Eye-catching featured content section with play/info buttons
- **Content Rows** — Horizontally scrollable movie/TV show carousels
- **Hover Previews** — Card enlargement and details on hover
- **Footer** — Complete with links and language selector

## 🛠️ Technologies Used

- **HTML5** — Semantic markup and accessibility
- **CSS3** — Flexbox, Grid, custom properties, and advanced animations
- **Responsive Design** — Mobile-first approach with media queries
